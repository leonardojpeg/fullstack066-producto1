# TechConnect

Aplicación web frontend para la gestión de **ofertas y demandas de empleo tecnológico**.

Este proyecto ha sido desarrollado como parte del módulo **Desarrollo Full Stack con JavaScript y Servicios Web**, en el que se implementa la interfaz de una plataforma colaborativa para conectar profesionales y empresas del sector tecnológico.
